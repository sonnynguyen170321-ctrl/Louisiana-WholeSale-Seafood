# Project Directory & Assets Layout

This rule describes the file registry and asset directories layout for the Louisiana Wholesale Seafood repository.

- **Image Assets:** All graphic assets, icons, logos, and banners must be stored inside the `assets/` subdirectory.
- **Root Elements:** Main structural files (`index.html`, `wholesale.html`, `restaurant.html`), main styles (`style.css`), and the interactivity script (`app.js`) must be kept in the project root directory.
