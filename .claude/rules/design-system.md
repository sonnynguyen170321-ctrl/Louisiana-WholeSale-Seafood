# Design System & Visual Guidelines

This rule outlines color tokens, typography rules, and responsive layout breakpoints.

### Color Palette
* **Primary (Crawfish Red):** `#EC1D27` (CSS Variable: `--color-primary`)
* **Accent (Gold Leaf):** `#E5C158` (CSS Variable: `--color-accent`)
* **Background Light (Warm Alabaster):** `#FEF9F3` (CSS Variable: `--color-bg-light`)
* **Background Dark (Deep Navy):** `#0A1118` (CSS Variable: `--color-bg-dark`)
* **Text Main (Soft Charcoal):** `#1E252D` (CSS Variable: `--color-text-dark`)
* **Text Light (Off-white):** `#F8F9FA` (CSS Variable: `--color-text-light`)

### Typography
* **Headings:** `'Cormorant Garamond', serif` (elegant serif style)
* **Body & UI Controls:** `'Inter', sans-serif` (clean modern sans-serif style)

### Responsive Breakpoints
* **Mobile Drawer Navigation:** `max-width: 768px` (Nav shifts to hidden list with sidebar toggles)
* **Tablet / Grid Adjustments:** `max-width: 1024px` (Columns stack vertically or reduce density)
* **Desktop Spacing:** `min-width: 1025px` (Full columns, widescreen spacing margins)
