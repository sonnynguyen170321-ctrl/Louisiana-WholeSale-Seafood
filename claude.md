# The BLAST Framework
- Blueprint: Initialize the project memory. Create your task_plan.md and project_constitution.md.
- Link: Connect your "Universal Remotes" (MCPs) like Superbase, Zapier, and NotebookLM.
- Architect: Build the logic. Parallel agents work here to create the front end and back end.
- Style: Apply the UI/UX Pro Max standards. No more boring white walls.
- Trigger: Deploy to the world via Vercel and set up automated schedules with Modal

# The NotebookLM Connection
- Install: Use the terminal script to link your "Brain" and authorize via the browser.
- Research: Programmatically trigger deep research to extract modules, scripts, and data tables.
- Execute: Automatically turn raw research into courses, newsletters, or competitor reports.
- Optimize: Use source_get_content to save 10x on tokens and build 10x faster.

# The UI/UX Pro Max Design Vault
- Skeleton: Define specific patterns like Bento Grids or SaaS Hero sections.
- Skin: Apply high-end aesthetics like Glassmorphism, Aurora UI, or the Linear Dark Mode.
- Soul: Add micro-interactions, "border beams," and staggered scroll reveals to wow users.
- Audit: Run the Pro Max Skill to fix SEO, accessibility, and 50+ UI flaws automatically.

# The Claude SWARM Framework
- Scope: Break the mission into right-sized tasks. Each agent gets a clear role, file boundaries, and success criteria. Too small = wasted overhead. Too large = agent overwhelm.
- Wire: Set up dependencies so agents auto-block until prerequisites are done. Use the shared mailbox for real-time coordination — no file I/O bottlenecks.
- Assign: Match the right model to the right job. Opus for team leads, Sonnet for builders, Haiku for validators. Model-mixing saves ~60% vs all-Opus.
- Run: Launch and monitor. Watch for stuck agents, file conflicts, and the main agent going rogue. Kill and reassign if tokens are burning without progress.
- Merge: Team lead compiles results, validators confirm quality, agents shut down. Log cost, save the prompt as a reusable template, and optimize for next time.
